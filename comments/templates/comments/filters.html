Filtro / Etiqueta Descripción Código de Plantilla Salida de Ejemplo
cut Elimina todas las ocurrencias del argumento en el string. {{ " AndrÉSCrUz -- 99 "|cut:" " }} AndrÉSCrUz! (Elimina espacios)
date Formatea un objeto de fecha/hora. {{ comments.0.date_posted|date:"D d M Y" }} {{comments.0.date_posted}}
now (Etiqueta) Obtiene la fecha y/o hora actual. {% now "jS F Y H:i" %} {{now}} (Hora actual aproximada)
lower Convierte un string a minúscula. {{ "AndrÉSCrUz"|lower }} andrés cruz!
upper Convierte un string a mayúscula. {{ "AndrÉSCrUz"|upper }} ANDRÉS CRUZ!
add Suma un valor (o concatena si es string). {{ "5"|add:"2" }} 7
with (Etiqueta) Crea una variable local temporal. {% with total=comments.0.id|add:5 %}{{ total }}{% endwith %} ({{comments.0.id}} + 5) ({{comments.0.id}} + 5)
cycle (Etiqueta) Alterna entre los valores proporcionados. {% for o in comments %}...{% endfor %} Para 3 elementos: r1, r2, r1
default Establece un valor por defecto si la variable es None o falsa. {{ comments.0.undefined|default:"No disponible" }} No disponible
length Devuelve el número de elementos (string o lista). {{ comments|length }} 15
safe Marca un string como seguro, previniendo el escape de HTML. {{ "Texto"|safe }} El output se renderiza como Texto (en negrita)
join Convierte una lista en un string con el delimitador. {{ comments|join:" - " }} ---
Filtros Anidados El resultado de cut es la entrada de lower. {{ "HOLA!!!"|cut:"!"|lower }} hola
